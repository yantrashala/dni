# dni

Node utility to **_download and extract (dni)_** a deployable module and install its dependencies.

# Installation

```sh
npm install dni -g
```

# Features

- Downloads a zipped(.tgz) deployable package.
- Unzips the downloaded package.
- Installs the dependencies from its package.json.

# Usage

```
$ dni <module_name>
```

# Example
```
$ dni npm-install-all
```
